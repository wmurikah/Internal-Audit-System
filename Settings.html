About Backup Modes

Realtime: Every write syncs to Sheets immediately. Best for redundancy, uses more API quota.

Incremental: Changes queue up and sync in batches on a schedule. Balances redundancy with performance.

Disabled: Sheets are not updated. All data lives in Firestore only. Use "Full Backup" to restore Sheets when needed.
